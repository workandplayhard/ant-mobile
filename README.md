# BoilerPlate

ReactNative boilerplate project

## Requirements

- You will need to confirm that your environment is correctly set up as described [here](https://reactnative.dev/docs/environment-setup)

- This app ran and tested on `node 16.13.2` and `nvm 0.39.1`.

- We use `yarn`

---

## Intro

This is a [React Native app](https://facebook.github.io/react-native/) for iOS and Android.

---

## Getting Started

#### 1. Clone and Install

```bash
# Clone the repo
git clone https://github.com/fatpig0416/rn-boilerplate.git

# Install node dependencies
yarn

# Install pods

cd ios && pod install
```

#### 2. Run app
```bash
yarn android
yarn ios
yarn iphone # `ios-deploy` package needs to be installed
```


#### 3. Setting up Bootsplash

You can follow [react-native-bootsplash](https://github.com/zoontek/react-native-bootsplash) documentation
